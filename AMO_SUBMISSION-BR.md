# LinkScout - Submissão Mozilla Add-ons

## Nome do Add-on
LinkScout

## Resumo (máximo 250 caracteres)
Salve múltiplos links de uma só vez com um único clique. Selecione texto contendo links e salve-os como favoritos organizados. Além disso, salve e feche todas as abas em uma ação para gerenciamento fácil de sessões.

---

## Descrição Completa (Português)

### 🔗 LinkScout - Salve Links de Forma Inteligente

**Pare de perder links importantes!** LinkScout é a extensão definitiva para Firefox que ajuda você a salvar múltiplos links de uma só vez com organização inteligente.

#### ✨ Funcionalidades Principais

**📌 Salvar Links da Seleção**
- Selecione qualquer texto contendo links em uma página
- Clique com botão direito e escolha "Salvar Links da Seleção"
- Todos os links são automaticamente extraídos e salvos como favoritos

**📑 Salvar e Fechar Todas as Abas**
- Salve todas as abas abertas em um único clique
- Abas são organizadas por data e hora (ex: "Segunda-feira 08/01/2026 15:30")
- Perfeito para gerenciamento de sessões e limpeza do navegador

**📁 Organização Inteligente**
- Links são organizados em pastas pelo título da página
- Estrutura: `LinkScout / Título da Página / Links`
- Abas são organizadas por data e hora da sessão

**⚙️ Configurações Personalizáveis**
- Escolha onde salvar favoritos (Barra, Menu ou Outros Favoritos)
- Defina um nome personalizado para a pasta raiz
- Ative/desative notificações

#### 🎯 Ideal Para
- Pesquisadores coletando múltiplas fontes
- Desenvolvedores salvando links de documentação
- Estudantes organizando materiais de estudo
- Qualquer pessoa que queira salvar múltiplos links rapidamente

#### 🔒 Privacidade
LinkScout funciona 100% localmente. Nenhum dado é enviado para servidores externos. Seus favoritos ficam no seu navegador.
